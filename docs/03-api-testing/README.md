# 🧪 API Testing

This section teaches how to test the microservices like a real backend or DevOps engineer, one API at a time.

## 🎯 Goal

By the end of this section, you will be able to:

- Test each API manually in Postman
- Use a Postman environment
- Run tests against direct service ports
- Run tests through the Nginx gateway
- Read failed test messages
- Prove your local stack works before CI/CD and AWS
