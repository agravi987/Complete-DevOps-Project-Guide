# Python API Test Templates

These are small beginner scripts for testing one API at a time.

Install dependency:

```powershell
py -m pip install requests
```

Run examples:

```powershell
py templates\python-api-tests\01_auth_health.py
py templates\python-api-tests\02_auth_register.py
py templates\python-api-tests\03_auth_login.py
```

Do not run everything at once while learning.
